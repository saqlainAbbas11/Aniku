## Aniku - Anime Discovery Web Application

# 📖 Overview
Aniku is a comprehensive anime discovery platform built with modern web technologies, providing users with seamless access to a vast database of anime content. From tracking currently airing series to exploring all-time favorites, Aniku offers an intuitive interface for anime enthusiasts to discover, explore, and stay updated with their favorite shows.

# ✨ Features
🔍 Advanced Search
Search any anime by title with instant results
Filter by genre, season, year, and rating
Auto-suggestions for faster discovery

# 📊 Comprehensive Content
Trending Anime - Discover what's popular right now
Currently Airing - Never miss a weekly episode
Top Anime - All-time fan favorites
Upcoming Series - Stay ahead of new releases
Manga Directory - Explore manga adaptations and originals

# 📱 User-Friendly Interface
Responsive design optimized for all devices
Dark/Light theme support
Smooth animations and transitions
Intuitive navigation with quick access to all sections

# 📈 Data Integration
Real-time data from JIKAN API (MyAnimeList's unofficial API)
Cross-platform compatibility with MAL, AniList, and Kitsu
Up-to-date episode information
Detailed anime pages with:
Synopsis and background
Episode lists and air dates
Ratings
